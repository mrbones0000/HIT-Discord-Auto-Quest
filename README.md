El autor no se hace responsable de baneos, bloqueos u otras sanciones derivadas del uso de este software.

**Descripción**

AutoQuest Selfbot es un bot de usuario desarrollado en Java que permite completar automáticamente:

Misiones diarias de Discord

Misiones o recompensas de juegos integrados en Discord

El bot automatiza tareas repetitivas, asegurando que las recompensas se obtengan de manera continua.

**Características**

Completa misiones automáticas de Discord

Soporte para misiones de juegos integrados

Automatización continua de tareas

Basado en librerías de Java compatibles con Discord

Recomendado ejecutar en Discord PTB o Canary para mayor seguridad

Compatible con múltiples servidores

**Requisitos**

Java 17 o superior

Maven o Gradle para manejar dependencias

Token de usuario (selfbot)

Discord PTB o Canary recomendado



El bot comenzará a completar automáticamente las misiones disponibles en Discord y los juegos integrados.
